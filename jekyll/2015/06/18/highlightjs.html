使用highlight.js做语法高亮

  • Published 18 Jun 2015
  • Category jekyll
  • Previous
  • Next

highlight.js支持多种语言和注释风格，非常优秀，推荐大家使用。

一、下载highlight.js

首先，登陆https://highlightjs.org/

可以点击live demo在线预览各demo的效果。

然后选择自己常用的语言，点击下载得到highlight.zip。

二、插入HTML代码

把styles文件夹和highlight.pack.js文件放在你的网站里，在你的_layouts中的html文件加入代码

<link rel="stylesheet" href="/path/to/styles/default.css">
<script src="/path/to/highlight.pack.js"></script>
<script>hljs.initHighlightingOnLoad();</script>

其中/path/to/替换成你的路径，default.css替换成你想要的风格。

三、插入代码时输入语言名

最后，在Markdown插入代码时，在注释符号之后加入语言名称即可看到效果。例如xml语言：

```xml
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" xml:lang="en" lang="en">
<head>
    <meta http-equiv="Content-Type" content="text/html: charset=utf-8" />
    <title>Page Title</title>
</head>
<body>
    <!-- body code here -->
</body>
</html>
```

效果如下
