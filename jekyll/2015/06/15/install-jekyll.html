在Ubuntu上安装Jekyll

虽然也有Windows版，但是Jekyll官方推荐在Linux上运行。以下是我的安装过程，仅作参考。

这里假设你已经成功安装了ubuntu，而且已经有了GitHub账号，创建了个人主页用的repository。

安装ruby-install

wget -O ruby-install-0.5.0.tar.gz https://github.com/postmodern/ruby-install/archive/v0.5.0.tar.gz

tar -xzvf ruby-install-0.5.0.tar.gz

cd ruby-install-0.5.0/

sudo make install

安装ruby 2.2.2

ruby-install ruby 2.2.2

因为ruby-install旨在同一台电脑上安装和管理多个不同版本的ruby，所以ruby-install并没有将ruby的路径加入环境变量PATH里。

需要加的童鞋可以自己将~/.rubies/ruby-2.2.2/bin加入PATH。

配置RubyGems的源

cd ~/.rubies/ruby-2.2.2/bin

./gem sources –remove https://rubygems.org/

./gem sources –a http://ruby.taobao.org/

安装jekyll

sudo apt-get install nodejs

./gems install jekyll

安装Git

sudo apt-get install git git-core git-gui git-doc git-svn git-cvs gitweb gitk git-email git-daemon-run git-el git-arch

配置Git

1. 检查ssh可用性

ssh –T git@github.com

如果看到：

The authenticity of host 'github.com (192.30.252.131)' can't be established. RSA key fingerprint is aa:bb:……:yy:zz:.

Are you sure you want to continue connecting (yes/no)? no

Host key verification failed.

则说明可以连接。

2. 创建本地ssh秘钥

检查~/.ssh目录下是否有idrsa（私钥）和idrsa.pub（密钥），如果有，则备份出来，删除源文件，再执行如下语句，没有则直接执行：

ssh-keygen –t rsa –C “你的GitHub登录用Email地址@gmail.com”

3. 登录GitHub设置公钥

在GitHub中登录你的账号，点击首页右上角”Settings”按钮，选择”SSH keys”→”Add SSH key”，将id_rsa.pub文件中内容复制进key栏中，title自定。

保存之后再次检查ssh连接情况：

ssh –T git@github.com

如果看到如下所示，则表示成功：

Hi LuckyStar04! You've successfully authenticated, but GitHub does not provide shell access.

4. Clone你的GitHub项目

有以下两种方式clone你的代码到本地：

git clone git@github.com:alioth310/test.git

git clone git://github.com/alioth310/test.git

开始享受jekyll吧！
