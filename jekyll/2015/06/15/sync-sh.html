用于Jekyll博客同步的小脚本

  • Published 15 Jun 2015
  • Category jekyll
  • Previous

因为通过Git更新Jekyll的步骤太繁琐了，所以写了一个小脚本，用于同步本地site到GitHub。

首先要在_config.yml中加上：

exclude: [luckystar04.github.com,sync.sh]

然后新建sync.sh文件，chmod改为755，输入以下脚本：

jekyll build

echo "输入本次修改的tag："

read tag

cp -rf ./_site/* ./luckystar04.github.com/

cd luckystar04.github.com

git add .

git commit -a -m "${tag}"

git push origin

大功告成！赶紧试一试。
